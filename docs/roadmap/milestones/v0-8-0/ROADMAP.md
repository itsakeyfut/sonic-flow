# v0.8.0 - 設定 & テーマ

## 概要

設定の永続化、テーマシステム、キーボードショートカット、メディアキー対応を実装し、日常使いに耐えるプレイヤーへ仕上げる。アルバムアート表示も統合する。

## ゴール

- 設定が TOML ファイルに保存され、再起動で復元
- ダーク/ライトテーマ + アクセントカラー カスタマイズ
- 主要操作のキーボードショートカット
- システムメディアキー対応 (再生/一時停止/次/前)
- アルバムアートが Now Playing に表示

## スコープ

### In Scope

- 設定永続化 (TOML)
- テーマシステム (ダーク/ライト + カスタムアクセント)
- キーボードショートカット
- システムメディアキー (Win / Linux / macOS)
- アルバムアート抽出 + 表示
- ウィンドウ状態の記憶

### Out of Scope

- 多言語対応 (v1.x)
- カスタムテーマ (CSS 的な)
- フルカスタマイズ (v1.x or プラグイン)

## リスク / 技術的検討事項

- **OS別 メディアキー**: 各 OS の API が大きく異なる。Linux MPRIS は仕様が複雑、macOS は dynamic linking 注意
- **アルバムアート**: 大量ファイルのサムネイル生成はI/O負荷高い。バックグラウンド + キャッシュ必須
- **テーマカスタマイズ**: Slint の global で表現できる範囲を見極める。動的な色変更は対応版次第
- **設定ファイル損傷**: TOML パースエラー時の回復戦略 (バックアップ + デフォルト復帰)
