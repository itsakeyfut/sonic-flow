## リスク / 技術的検討事項

- **クロスプラットフォーム差**: 各 OS のオーディオサブシステムで挙動差がある (CoreAudio / WASAPI / ALSA / PulseAudio / Pipewire)
- **GPU ドライバ依存**: AMD / NVIDIA / Intel でシェーダー動作が異なる可能性
- **長時間動作のリーク**: 短時間テストでは見えないリークがある。24h テストは必須
- **メモリ目標 (200MB active) の難易度**: アルバムアートやライブラリキャッシュで膨らみやすい。リーンに保つ工夫が必要
